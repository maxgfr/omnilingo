## 1. AI-Centric Learning

### 1.1 Smart Daily Session
- [ ] **AI-powered daily session** — When the user opens the app, show a personalized "Today's session" screen that combines:
  - SRS cards due for review
  - New words selected by AI based on weak areas (from errors table)
  - A mini grammar exercise on a topic the user struggles with
  - The whole session is generated by the AI tutor at start, not random
- [ ] **Adaptive word selection** — Instead of random unlearned words, use AI to pick words that:
  - Build on vocabulary the user already knows (semantic clusters)
  - Match the user's weak categories (from frequent errors)
  - Progress naturally through themes (food → restaurant → ordering)
- [ ] **AI exercise generation** — After each grammar lesson, the AI generates 5 custom exercises tailored to the user's mistakes
- [ ] **Conversation mode** — Guided conversation practice where the AI plays a role (waiter, colleague, friend) and evaluates the user's responses
- [ ] **AI pronunciation feedback** — After STT transcription, send both the expected text and the transcript to the AI for detailed pronunciation tips (not just match/no-match)

### 1.2 AI Model Configuration
- [ ] **Extended provider list** — Add providers:
  - Codex (OpenAI Codex CLI — local, subprocess like Claude CLI)
  - Claude Code (claude-code subprocess — local, no API key)
  - GLM free tier / paid tier distinction
  - Ollama (local LLM — configurable URL)
  - Custom OpenAI-compatible endpoint (user enters URL + key)
- [ ] **Model presets** — Quick select: "Fast & cheap" (haiku/flash), "Balanced" (sonnet/4o-mini), "Best quality" (opus/4o)
- [ ] **Per-task model assignment** — Different model for different tasks:
  - Chat: use the best model (opus/4o) for quality conversations
  - Exercise generation: use a fast model (haiku/flash) for speed
  - Translation: use a specialized model
  - UI translation: use the cheapest model
- [ ] **Test connection button** — Verify API key works before saving
- [ ] **Token usage tracking** — Track and display API call count + estimated cost per session
- [ ] **Offline fallback** — When no API key configured and no Claude CLI available, use built-in exercises only (no AI features disabled gracefully)
